# n8n-nodes-kavita

[![npm version](https://img.shields.io/npm/v/n8n-nodes-kavita.svg)](https://www.npmjs.com/package/n8n-nodes-kavita)

n8n community node for Kavita (comics/manga server) via its API

Install via **Settings -> Community Nodes -> Install** -> `n8n-nodes-kavita`.
